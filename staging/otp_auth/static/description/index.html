Website OTP Authentication.

OTP Authentication for secure Signup/Login Process in Odoo Website!

Odoo Website OTP Authentication provides a two-layer authentication feature. A very helpful module which adds an extra layer of security and provides an option to log-in/sign-up via One Time Password.

Account Security is an irrefutable subject when it comes to protecting the data in your Odoo Website. The OTP Authentication saves your Odoo Website from replay attacks; hence, avoids the unauthorized login and signup.

Apart from security the One Time Password Authentication facilitates an ease to your customers so that they can login to their account without actually remembering the password.

Making your Odoo website a safer place, now you can put the one time password authentication for login and sign up process. Odoo Website OTP Authentication one-time password facility to the Odoo website. Now your customers have the choice to sign in using a password or an OTP which is sent to their registered email address.

Set the One Time Password Authentication for your Odoo Website.

Now, as an admin you have the right to set up OTP Authentication for your Odoo Website. You can set it for your customers for the time of login as well as signup.

It just not offers security to customers accounts but also offers an ease of entering their Odoo Website Account easily without even remembering the password.

The customers can easily get the One Time Password to authenticate themselves for login or signup process in their registered e-mail id.

Set your own terms for the OTP Authentication in Your Odoo Website.

As an admin you have complete authority to set the configuration for One Time Password Authentication in your Odoo Website as you want.

The module allows you to enable or disable the OTP Authentication in your website anytime. Moreover, you can also enable/disable the OTP authentication individually for Signup and login.

Apart from that, you can also set the time limit expiry of OTP and also set the OTP type as you want.

Detailed Features List
Below is the detailed list of Features.
Set OTP Authentication for your Odoo website
  • You can set the OTP Authentication for login.
  • Also, set the OTP configuration for Sign in Authentication of your.
Customers will receive OTP in e-mail.
  • The customer’s email address is validated with the OTP during signing up on the Odoo website.
  • Hence, The OTP is sent to the customer’s registered email address.
Customers choice to Use OTP Authentication.
  • Customers can choose to log in to his Odoo website account with an OTP instead of the password.
Configure the OTP Authentication as you want.
  • The Odoo Admin can set expiry time of the OTP Authentication.
  • Also, set the OTP type (password or text) for the OTP.
  • It also allows the Odoo Admin to configure the email template of the sent OTP.
Maintain the Data in Odoo Backend.
  • The OTP validation information along with their success status will be maintained in the backend.
Cross-browser Compatibility.
  • Odoo Website OTP Authentication module is cross-browser compatible.
