Moyasar Payment Gateway

This module will plugin the Moyasar Payment Gateway with Odoo, so the customer will be able to use the Moyasar payment acquirer in the payment method for website.

Moyasar payment method will provide you with an option of Card payment, STC Pay, and Apple pay.

Main Features

  • Cards Payment
  • StcPay
  • ApplePay

Configuration For Moyasar Payment Gateway

Go to Website > Configuration > Payment Providers

Also you have to activate Moyasar.

Configure credentials and enable payment method.

Choose Moyasar method.

Fill popup for the payment.

Moyasar also provides ApplePay option.

As per Moyasar, Apple Pay is available on the following platforms:

  • Safari iOS 10 and later
  • Safari macOS 10.12 and later

If payment is success, then you will see confirmation page with suceess message.

If payment Fails, then you will see confirmation page with Failed message.

NOTE: You will find the full documentation in the module folder.
